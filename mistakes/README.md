# ⚠️ The Failure Audit
"The man who makes a mistake and doesn't correct it, is making another mistake."

Every time I get a Wrong Answer (WA), Time Limit Exceeded (TLE), or Runtime Error (RE), I record it here to ensure it never happens again.

## 🔥 Critical Lessons
1. **Integer Overflow:** Always check if `n > 2*10^9`. Use `long long`.
2. **Precision:** Use `long double` for geometry, never `float`.
3. **Array Bounds:** Always check `n+1` for 1-based indexing.
